PARTS OF THE COMPUTER

Home Parts Contact Me!

Monitor : The screen that displays your computers output, allowing you to see text, images, and videos.

Keyboard : An input device used to type letters, numbers, and commands into the computer.

Mouse : A pointing device that allows you to interact with the computer's graphical user interface by moving a cursor on the screen and clicking on icons and buttons.

Speakers : Output devices that produce sound, allowing you to hear music, videos, and other audio from your computer.

SSD : A type of storage device that uses flash memory to store data, providing faster read and write speeds compared to traditional hard drives.

HDD : A traditional storage device that uses spinning disks to read and write data, typically offering larger storage capacities at a lower cost compared to SSDs.

RAM : A type of computer memory that temporarily stores data and instructions that the CPU needs to access quickly while performing tasks.

PSU : A component that converts electrical power from an outlet into a form that the computer can use, providing power to all the other components.

Motherboard : The main circuit board that connects all the components of the computer, allowing them to communicate with each other.

CPU : The central processing unit, often referred to as the "brain" of the computer, responsible for executing instructions and performing calculations.

CPU Cooler : A device that helps dissipate heat generated by the CPU, preventing it from overheating and ensuring optimal performance.

GPU : A specialized processor designed to handle graphics rendering and complex calculations, often used for gaming, video editing, and other graphics-intensive tasks.
